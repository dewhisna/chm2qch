CHM2QCH - Converts Windows CHM files to Qt Help format.

This program extracts all contents from CHM file to target directory, creates Qt Help Project file and runs qhelpgenerator

Command line usage: chm2qch [options] input

Options:
-?, -h, --help Displays help.
-g, --generate Run qhelpgenerator to produce QCH file
-r, --no-root Do not write root contents section
-q, --quiet Quiet mode. Do not write any messages to stdout
-n, --namespace <name>Set documentation namespace to <name>
-c, --clean Delete intermediate files after running qhelpgenerator (use with -g)
-d, --directory <dir> Set target directory to <dir>
-b, --qtbindir <dir> Set Qt binaries location to <dir>
Arguments:
input  Input CHM file
